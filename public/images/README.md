# Imágenes de Jaguar App

## Estructura de Carpetas

```
public/images/
├── README.md
├── ejemplares/      # Imágenes de ejemplares individuales
├── galeria/          # Imágenes para la galería general
├── hero/             # Imágenes para la sección hero
└── iconos/           # Iconos y elementos gráficos pequeños
```

## Cómo agregar imágenes

1. Coloca tus imágenes en la carpeta correspondiente
2. Usa rutas relativas desde `public/images/` en tu código
3. Ejemplo: `/images/ejemplares/jaguar1.jpg`

## Formatos soportados

- JPG/JPEG
- PNG
- SVG
- WebP
